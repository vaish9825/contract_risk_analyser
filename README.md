# ClauseRisk Analyzer

ClauseRisk Analyzer is a legal intelligence tool designed to help **non-lawyers, freelancers, and SMEs** understand contract risks instantly. It analyzes uploaded contract PDFs and highlights risky clauses with clear, plain-English explanations.

Built as a hackathon MVP, the project demonstrates how modern AI can simplify legal review and reduce hidden contractual risks.

Live Demo:
https://clause-risk-analyser.streamlit.app

---

## 🔍 What It Does
- Upload a PDF contract
- Automatically extracts and analyzes clauses
- Flags risky terms such as uncapped liability, auto-renewals, and ambiguous termination clauses
- Presents results through an interactive risk dashboard

---

## 🧠 How It Works
- **Frontend:** Lightweight Streamlit web app
- **PDF Processing:** Python-based text extraction
- **AI Engine:** LLM-powered clause extraction and risk analysis
- **Output:** Clause-level risk categorization with explanations

A 10-page contract can be analyzed in **2–3 minutes**.
